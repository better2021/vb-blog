# Markdown 博客文章示例

这是一篇 Markdown 格式的博客文章示例。实际发布文章时，请把这个文件复制到 `content/posts/` 目录，并将文件名改成英文 slug，例如 `my-first-post.md`。

## 写作建议

- 标题保持清晰，方便读者快速判断主题。
- 摘要控制在一两句话内，用来展示在首页列表中。
- 标签数量不必很多，但应该稳定复用。

## 代码示例

```js
function greet(name) {
  return `你好，${name}`;
}
```

Markdown 可以让技术笔记保持干净、可读，也方便后续迁移到其他博客系统。
